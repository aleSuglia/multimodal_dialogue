The game

You are one click away from starting the multimodal dialogue game, a variant of I spy with my little eye. It is two-player guessing game where you will be either assigned the spy or questioner role. As a spy you will be assigned an object in a scene, while the questioner can ask yes/no questions to figure out which object that is. Please check out the side bar for for more information about the rules.
